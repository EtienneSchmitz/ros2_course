Cette section décrit, étape par étape, comment monter une machine de travail prête pour le cours : système, ROS 2, simulateur, workspace.

## Pré-requis matériels

- Ordinateur capable de virtualiser une scène Gazebo Ionic (recommandé : **8 Go de RAM**, **CPU 4 cœurs**, **20 Go libres** sur disque).
- GPU dédié optionnel mais recommandé pour Gazebo et RViz fluides.
- Clé USB d'au moins **8 Go** (pour l'installation Ubuntu).
- Connexion Internet pour les téléchargements de paquets et de repos.

## Stack cible

| Couche | Version |
| --- | --- |
| Système | Ubuntu 24.04 LTS (Noble Numbat) |
| ROS 2 | Kilted Kaiju |
| Simulateur | Gazebo Ionic |
| Outils workspace | vcstool, rosdep, colcon |
