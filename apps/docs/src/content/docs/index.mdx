## Programme — 5 jours

<CardGrid>
  <Card title="Installation" icon="setting">
    Mise en place de l'environnement ROS 2 Kilted, simulateur Gazebo Harmonic, et packages robots maison (Kiwi + SO-101).
  </Card>
  <Card title="Jour 1 — Introduction" icon="open-book">
    Concepts ROS 2, nodes, topics, services, actions. Premier graphe en simulation.
  </Card>
  <Card title="Jour 2 — Navigation" icon="rocket">
    SLAM et navigation autonome avec la base mobile Kiwi (Nav2 + LIDAR).
  </Card>
  <Card title="Jour 3 — Manipulation" icon="puzzle">
    Pick & place avec le bras SO-101, MoveIt 2, planification de trajectoire.
  </Card>
  <Card title="Jour 4 — Vision" icon="magnifier">
    Perception, OpenCV, détection d'objets et asservissement visuel.
  </Card>
  <Card title="Jour 5 — Intégration" icon="approve-check">
    Mise en commun navigation + manipulation + vision sur un scénario complet.
  </Card>
</CardGrid>
