# 🎯 Objectifs

Cette première journée pose les bases du workshop. L’objectif est de vous donner une vision claire de l’environnement ROS 2 et de vous rendre opérationnels rapidement pour la suite.

## 🧭 Contenu de la journée

- Présentation générale de ROS 2 et de son écosystème
- Explication des concepts fondamentaux
- Installation ou vérification de l’environnement
- Exploration des commandes de base pour interagir avec ROS 2
- TP : création d’un package simple + exécution d’un nœud de test, etc...

## 🎓 À la fin de cette séance, vous serez capables de :

- Comprendre ce qu’est ROS 2, son rôle et ses cas d’usage
- Différencier les concepts clés : **nœud**, **topic**, **service**, **action**, **package**
- Savoir comment est structuré un projet ROS 2
- Créer, compiler et exécuter un **package minimal** en Python.
- Interagir avec un système ROS 2 via la ligne de commande (`ros2 topic`, `ros2 run`, etc.)
- Préparer votre environnement de travail pour les jours suivants
